Войди в роль учителя Python. Мне нужно создать курс Python для начинающих. Темы для курса:

1. Введение в Python. Установка Python в Windows. Пишем первую программу на Python и печатаем Hello world.
2. Назначение базовых переменных. Изучение того, как назначать переменные с примерами
3. Строки. Узнайте все о строках Python. Выясните, как автоматически создавать, переупорядочивать, переназначать, разбирать и собирать блоки текста!
4. Целые числа. Узнайте все о числах в Python
5. Условия (if, else, elif). Узнайте, как выстраивать условия в Python, включив операторы if, else и elif. Изучение логических переменных и логических операторов.
6. Списки. Узнайте о списках — структуре данных в Python, используемой для хранения упорядоченных групп данных.
7. Словари. Узнайте все о структуре словаря Python и о том, как создавать и использовать пары ключ-значение в своем коде.
8. Циклы. Циклы — это структуры, которые позволяют повторять код Python снова и снова. Научитесь читать циклы и писать их для решения собственных проблем.
9. Функции. Узнайте о повторном использовании кода с функциями Python
10. Промежуточная сертификация (Решение задач с применением полученных знаний во всех модулях)


Каждая тема — это лекция. В лекциях должны быть примеры кода для лучшего понимания. После основного текста лекции должно быть пять вопросов по теме лекции. Запомни все темы, потому что они понадобятся тебе позже. Начнем с четвёртой темы. Напиши только четвёртую лекцию следуя правилам. Объем четвёртой лекции должен составлять от 1000 до 1200 слов. Не забудь написать об основных методах типа данных integer в python, приведи примеры к методам. Помни, что примеры очень важны и их надо привести как можно больше. Каждая лекция должна включать текст для обучающегося и десять задач, чтобы закрепить понимание студента. Задачи не должны быть слишком простыми. Каждая задача должна иметь входные данные и ожидаемые выходные данные. Сложность заданий увеличивается с каждым заданием.
